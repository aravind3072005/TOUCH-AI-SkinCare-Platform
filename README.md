# TOUCH-AI-SkinCare-Platform
Initial release of the TOUCH AI Skincare Platform featuring AI-powered skin analysis, personalized skincare recommendations, chatbot assistance, secure user authentication, backend database integration, and an online skincare product catalog.



**# 🧴 TOUCH AI Skincare Platform

An AI-powered skincare platform designed to help users understand their skin condition through intelligent analysis, receive personalized skincare recommendations, interact with an AI skincare assistant, and purchase recommended skincare products through an integrated online shopping system.

---

## 🌟 Project Overview

TOUCH AI Skincare Platform combines Artificial Intelligence, Web Development, and Database Management to create a complete online dermatology solution.

The platform analyzes a user's skin condition, recommends suitable skincare products, provides chatbot-based assistance, and allows users to securely purchase dermatologist-recommended products.

## 🚀 How to Run

1. Clone the repository

```
git clone https://github.com/aravind3072005/TOUCH-AI-SkinCare-Platform.git
```

2. Move the project into your XAMPP `htdocs` folder.

3. Start Apache and MySQL using XAMPP.

4. Import the MySQL database.

5. Open your browser and visit:

```
http://localhost/TOUCH-AI-SkinCare-Platform
```
